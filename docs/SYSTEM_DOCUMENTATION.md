## 产品概述

### 产品定位

Page Cleaner 是一款 Chrome 浏览器扩展，帮助用户：
- 在干净的侧边栏中阅读网页内容
- 使用 AI 自动分析网页内容
- 支持多种页面类型（文章、技术文档、食谱等）

### 目标用户

- 需要阅读大量网页内容的用户
- 需要快速获取网页摘要的用户
- 对网页广告和干扰元素敏感的用户

### 版本信息

- 版本号：1.0.0
- Manifest 版本：3
- 支持浏览器：Chrome 88+

## 核心功能

### 1. 页面清理 (Clean Page)

**功能描述**：提取网页主要内容，去除广告、导航栏、侧边栏等干扰元素。

**支持页面类型**：
| 类型 | 说明 |
|------|------|
| `article` | 普通文章页面 |
| `technical_article` | 技术文章（含代码块） |
| `docs_page` | 文档页面 |
| `recipe` | 食谱页面 |
| `feed_page` | 列表/首页（部分支持） |

**处理流程**：
1. 获取页面 HTML
2. 检测页面类型
3. 选择提取策略
4. 提取可读内容
5. 格式化输出

### 2. AI 分析 (AI Analysis)

**功能描述**：使用 AI 模型分析网页内容，生成摘要、大纲、关键点等。

**分析内容**：
- 文章摘要
- 内容大纲
- 关键引用
- 核心要点
- 代码笔记（技术文章）

**支持的 AI 提供商**：
- OpenRouter
- GLM-5（智谱 AI）
- 自定义服务器

### 3. 订阅系统 (Subscription)

**功能描述**：通过 PayPal 订阅解锁无限使用。

**订阅计划**：
| 计划 | 价格 | 周期 |
|------|------|------|
| Pro Monthly | $4.99 | 月付 |
| Pro Yearly | $39.90 | 年付（省 33%） |

**订阅状态**：
- `active` - 活跃订阅
- `canceled` - 已取消（当前周期结束前仍有效）
- `expired` - 已过期
- `inactive` - 未激活

### 4. 用户认证 (Authentication)

**功能描述**：使用 Supabase 进行用户认证。

**认证方式**：
- 邮箱 + 密码注册/登录
- Demo 账号：`demo@example.com` / `demo123`

### 5. 访客限制 (Guest Limit)

**功能描述**：未登录用户有使用次数限制。

**限制规则**：
- 免费使用次数：3 次
- 计数存储：Chrome Storage / localStorage
- 超出限制后提示登录

## 部署指南

### 本地开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

### Chrome 扩展打包

1. 运行 `npm run build`
2. 打开 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `dist/` 目录

### Chrome Store 发布

1. 准备素材：
   - 扩展图标（16x16, 48x48, 128x128）
   - 宣传图（440x280, 920x680）
   - 截图（1280x800 或 640x400）

2. 打包扩展：
   ```bash
   cd dist
   zip -r ../page-cleaner-extension.zip .
   ```

3. 上传到 Chrome Developer Dashboard
